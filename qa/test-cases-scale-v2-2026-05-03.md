# Масштабный Набор Тест-Кейсов (v2)

Дата: 2026-05-03  
Продукт: `https://timur-ship-it.github.io/logist-city/`  
Цель: полное покрытие критического пользовательского пути от выбора позиций до выгрузки/отправки заявки.

## Правила прогона
- Приоритеты: `P0` (блокер отправки), `P1` (критично), `P2` (важно), `P3` (косметика).
- Статусы: `Not Run`, `Pass`, `Fail`, `Blocked`.
- Для каждого `Fail` фиксировать: шаг, фактический результат, скрин, товар/артикул.
